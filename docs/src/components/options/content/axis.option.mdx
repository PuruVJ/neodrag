---
title: axis
type: "'both' | 'x' | 'y' | 'none'"
defaultValue: "'both'"
---

Axis on which the element can be dragged on. Valid values: `both`, `x`, `y`, `none`.

- `both` - Element can move in any direction
- `x` - Only horizontal movement possible
- `y` - Only vertical movement possible
- `none` - No movement at all
