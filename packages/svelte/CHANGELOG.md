If you have this in your `tsconfig.json`:

```json
{
	"compilerOptions": {
		"types": ["@neodrag/svelte/globals"]
	}
}
```

or in your `.d.ts` file:

```ts
/// <reference types="@neodrag/svelte/globals" />
```

Remove it. It is no longer needed.
